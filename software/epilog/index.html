EpiLog

EpiLog is a Java software for the qualitative simulation of epithelial patterning.

It defines an epithelium as a 2D grid of hexagonal cells, each containing a LRG regulating the internal proper components. It is also capable of integrating as input components of each cell, stable environmental influences as well as varying signals from neighbouring cells. The software provides a graphical user interface representing the hexagonal grid of cells to facilitate the definition of initial conditions, definition of perturbations, and visualization of the simulation. The latter is performed synchronously with respect to all components of all cells.

It relies on bioLQM to import multi-valued logical models from SBML qual and compute the successor states.
