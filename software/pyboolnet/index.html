PyBoolNet

PyBoolNet is a Python package for the generation, modification and analysis of Boolean networks. The current version is available at https://github.com/hklarner/PyBoolNet/releases.

For the manual, a reference and tutorials see the PyBoolNet manual. For questions and suggestions do not hestitate to open issues or contact hannes.klarner at fu-berlin.de

PyBoolNet - A Python Package for the Generation, Analysis and Visualisation of Boolean networks. H. Klarner, A. Streck, H. Siebert. (under review).
