mpbn

Python and Answser-Set Programming implementation for reachability analysis and listing of attractors and reachable attractors using Most Permissive Boolean networks.

See documentation and notebook examples.

Reconciling qualitative, abstract, and scalable modeling of biological networks. L. Paulevé, J. Kolčák, T. Chatain, S. Haar. Nature Communications 11 (2020).
